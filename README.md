# 朋克周长文

AI科技公众号长文写作skill。

## 风格DNA

**公路商店的态度。** 松弛的不在乎，不是用力的愤怒。

**槽边往事的洞察。** 那种"大家都感觉到了但没人说出来"的东西。

**小约翰可汗的幽默。** 正经事不正经讲，信息密度高但不累。

## 核心原则

- **第一人称亲历**：不是写报道，是写体验
- **具体打败抽象**：用一个具体画面替代抽象概念
- **态度从内容长出来**：有信息量的态度才是好态度
- **一篇文章只讲一个核心洞察**：不贪多
- **信息增量**：别人写过的角度不要写
- **开头零废话**：第一句话就要让人停下来

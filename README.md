# 朋克周长文

AI科技公众号长文写作skill。

## 风格DNA

**开篇即判断。** 第一句话就是态度或判断，不铺垫，不绕弯。

**比喻密集。** 每两三段就有一个比喻，荒谬但合理。

**段落极短。** 一句话一段是常态，节奏极快。

**幽默有信息量。** 段子是推进内容的武器，不为搞笑而搞笑。

**态度从真实体验长出来。** 每句带态度的话背后都有真实场景支撑。

**敢得罪人。** 直接点名公司、产品、人，不怕被投诉。

## 核心原则

- **开篇即判断**：第一句话就是态度，不"今天我想聊聊"
- **比喻密集**：每两三段至少一个比喻，荒谬但合理
- **段落极短**：一句话一段是常态
- **幽默有信息量**：每个段子背后都有信息
- **真实体验支撑**：先写你做了什么，再写判断
- **敢得罪人**：直接点名，不绕弯
